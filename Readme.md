# VS Solutions Parser 
Dieses Projekt soll Visual Studio Solution-Dateien lesen und schreiben. 

Leider unterstützen der MSBuild-Workspace oder der AdHoc Workspace aus dem Roslyn Projekt nicht das erstellten von Visual Studio Projekt-Dateien von außerhalb von Visual Studio.

##Funktionsumfang 
- Laden von Solution-Dateien von Visual Studio 2015 und 2017 
- Die Projekt werden in einem Objektmodell zur Verfügung gestellt.
- 

##Offene Punkte 
* Unsterstützung von anderen ProjektTypen wie z.B. Web-Projekte, VB.NET, ...
